pandocがインストールされていない場合はgithubリポジトリからインストール

https://github.com/jgm/pandoc/

```bash
wget https://github.com/jgm/pandoc/releases/download/3.1.11.1/pandoc-3.1.11.1-1-amd64.deb && sudo dpkg -i pandoc-3.1.11.1-1-amd64.deb
```
